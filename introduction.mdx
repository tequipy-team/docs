The Tequipy Integration API lets you connect your HR and IT systems to Tequipy to
manage orders, users, equipment, offboardings, collections, redeployments, and
invoices programmatically.

## Base URL

All requests are made to:

```
https://prod.backend.tequipy.com/integration
```

## Authentication

Every endpoint requires a Bearer JWT token. Message us in your client Slack channel or email support@tequipy.com to obtain
your Bearer token.

## Conventions

List endpoints are paginated and accept `page`, `size`, and `sort` query
parameters. Query endpoints accept a JSON body for filtering and return a paginated
result set.
